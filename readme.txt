== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin dashboard
2. Navigate to **Plugins > Add New**
3. Search for "Suffix Master"
4. Click **Install Now** and then **Activate**
5. Go to **Suffix Master** in your admin menu to configure

= Manual Installation =

1. Download the plugin zip file
2. Upload to `/wp-content/plugins/suffix-master/`
3. Activate the plugin through the **Plugins** menu
4. Configure settings at **Suffix Master** in your admin menu

= Quick Setup =

1. **Global Settings**: Configure default suffixes in **Global Suffix Settings**
2. **WooCommerce**: Set up product and price suffixes in **WooCommerce Settings**
3. **Per-Item**: Override settings using metaboxes on individual posts/pages/products
4. **Preview**: Enable live preview to see changes in real-time

== Frequently Asked Questions ==

= Does this work with WooCommerce? =

Yes! Suffix Master has comprehensive WooCommerce integration including product titles, prices, and slugs. It works with all WooCommerce themes and extensions.

= Can I override global settings for specific items? =

Absolutely! Every post, page, and WooCommerce product has a "Suffix Master Settings" metabox where you can set custom suffixes that override global settings.

= What placeholders are available? =

Currently supported: `{year}` (current year) and `{site_name}` (WordPress site name). More placeholders will be added in future updates.

= Will this affect my SEO? =

When used properly, Suffix Master can enhance your SEO by creating consistent, keyword-rich titles and URLs. The slug suffix feature is particularly useful for SEO optimization.

= Is it compatible with caching plugins? =

Yes, Suffix Master is compatible with all major caching plugins including WP Rocket, W3 Total Cache, and WP Super Cache.

= Can I use HTML in content suffixes? =

Yes, the content footer suffix field supports HTML, allowing you to add formatted text, links, and styling to your content endings.

= Does it work with custom post types? =

Currently, Suffix Master supports posts, pages, and WooCommerce products. Support for custom post types is planned for future releases.

= How do I reset all settings? =

Use the "Reset All Settings" button in the **Tools & Actions** tab. This will clear all configured suffixes and restore defaults.
